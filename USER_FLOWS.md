## Marker System

### Physical Markers

**Option A: QR Code Labels**
- 3" x 3" laminated QR codes
- QR encodes: `zonecount://zone/{zone_id}` or `zonecount://site/{site_id}/zone/{code}`
- Include human-readable zone code below QR (e.g., "WIC-01")

**Option B: Text Labels (OCR)**
- Bold, high-contrast text labels
- Format: `[ZONE CODE]` in large text
- Example: "DRY-01" in 72pt bold
- Simpler to produce, no special printing needed

**Recommended Placement:**
- Walk-ins: On door at eye level
- Dry storage bays: On shelf end-cap
- Reach-ins: On door handle area
